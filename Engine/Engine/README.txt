ENGINE FOR CREATING A TEXT GAME (QUEST)
USER MANUAL

To create a text game with this engine, you should do only two steps:
1. Create your own files with sounds (sound files should be .WAV format), main texts (scenes) and texts that will be shown on control buttons. 
2. Put these files to corresponding folders. Sound files in X:\Engine\Engine\Sounds, 
scenes in X:\Engine\Engine\Texts and names of control buttons in X:\Engine\Engine\Texts\Text_for_buttons.
========================================================================================================================
NOTE: You should create special files named "Text_default.txt" (it should be in the same folder as texts for buttons),
"Scene_default.txt" (it should be in the same folder as main texts) and
"Sound_default" (it should be in the same folder as sounds). They are use if some main files were lost.




ДВИЖОК ДЛЯ СОЗДАНИЯ ТЕКСТОВОЙ ИГРЫ (КВЕСТ)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Чтобы создать текстовую игру с этим движком, вы должны сделать всего два шага:
1. Создайте свои собственные файлы со звуками (звуковые файлы должны быть .WAV-форматом), основными текстами (сценами) и текстами, которые будут отображаться на кнопках управления.
2. Поместите эти файлы в соответствующие папки. Звуковые файлы в X: \ Engine \ Engine \ Звуки,
Сцены в X: \ Engine \ Engine \ Тексты и имена кнопок управления в X: \ Engine \ Engine \ Texts \ Text_for_buttons.
================================================== ================================================== ====================
ПРИМЕЧАНИЕ. Вам следует создать специальные файлы с именем «Text_default.txt» (он должен быть в той же папке, что и тексты для кнопок).
«Scene_default.txt» (он должен находиться в той же папке, что и основные тексты) и
«Sound_default» (он должен быть в той же папке, что и звуки). Они используются, если некоторые основные файлы были потеряны.
